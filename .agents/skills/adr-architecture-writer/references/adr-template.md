# ADR Template

```markdown
# ADR-NNN: Title

## Status

Proposta | Aceita | Substituida | Rejeitada

## Contexto

What forces, constraints, and facts require a decision?

## Decisao

What are we choosing?

## Alternativas Consideradas

- Option A: reason rejected.
- Option B: reason rejected.

## Consequencias

- Positive consequence.
- Negative consequence.
- Operational or testing impact.

## Validacao

How will implementation prove the decision works?
```
